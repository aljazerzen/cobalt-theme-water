Simple

With minimal visual clutter, the main focus is on the content of your site. here is also no JavaScript, no sea of CSS files and no heavy images.

Semantic HTML

This theme produces readable HTML, without classes and inline styles. All styling is provided by a single file: water.css

Adaptive

Honors system preference for light/dark theme. Fits well to all screen sizes, layouts and zoom levels.
